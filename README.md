# StadiumSync — Live Stadium Intelligence Platform

StadiumSync is a smart venue management and fan experience platform designed to revolutionize how stadiums operate during high-traffic events like IPL matches.

## 🚀 Overview
StadiumSync provides real-time venue intelligence, merging crowd density analytics, queue management, and AI-driven fan assistance into a unified dashboard. Built with a focus on premium aesthetics and responsive performance.

## ✨ Key Features
- **Live Heatmap:** Real-time crowd density visualization across 12 venue zones.
- **Smart Queue Tracking:** Live wait times for entry gates, food courts, and facilities.
- **AI Assistant:** An intelligent bot (powered by Claude) providing in-venue guidance, navigation, and live updates.
- **3rd Umpire View (CCTV):** Computer-vision enabled monitoring of crowding and entry flow.
- **Mission Control Dashboard:** A comprehensive view of venue health, safety scores, and revenue analytics.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Analytics:** Chart.js
- **Mapping:** Leaflet.js / Custom Canvas-based Venue Map
- **Icons:** Material Icons Round
- **Typography:** Inter & JetBrains Mono

## 📦 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/iusgupta20/stadium-sync.git
   ```
2. Open `index.html` in any modern web browser.
3. To enable the AI Assistant, enter your Anthropic API Key in the **Settings** tab.

## 🛡️ Privacy & Security
This repository has been sanitized for public release. All personal identifying information (PII) has been removed. API keys are handled securely in local memory only.
